# Coding Questions Challenge

This repository serves as a collection of my solutions to various programming problems, algorithms, and data structure challenges. 

## 📁 Repository Structure

- **`LeetCode/`**: Contains solutions to algorithmic problems from LeetCode. Each folder generally corresponds to a specific problem (e.g., `TwoSumProblem`).
- **`pom.xml`**: A Maven configuration file set up to compile the codebase using Java 17. 

## 🚀 Getting Started

### Prerequisites
- **Java Development Kit (JDK) 17** or higher.
- **Maven** (optional, but recommended for building the project).

### Setup Instructions

1. **Clone the repository:**
   ```bash
   git clone <your-repository-url>
   ```
2. **Open in your IDE:** 
   You can open this project directly in an IDE like IntelliJ IDEA or VS Code. The IDE should automatically recognize the `pom.xml` file and configure the Java 17 SDK for you.
3. **Compiling the code via Terminal:**
   Navigate into the project root directory and run:
   ```bash
   mvn clean compile
   ```
   *(Note: The `pom.xml` is configured to look within the root directory for Java packages instead of the standard Maven `src/main/java` directory).*

## 💡 About The Solutions

- Focus is kept on optimizing **Time** and **Space Complexity**.
- Most solutions are written as simple functions encapsulated in standard Java classes.
